Step 1: Wordlist Selection

Directory busting ka sabse bada raaz "Wordlist" hai. Agar wordlist me directory ka naam nahi hai, toh tool use nahi dhund payega. Hum common.txt se start karenge.

Seclists Dirb Lists RockYou (Not for dirs)

Step 2: Running the Attack

Ffuf tool ka use karke hum website ke har folder par "Brute-force" karenge. Dhyan rahe hum extensions (.php, .config) bhi check karein.

ffuf -u http://target.com/FUZZ -w /usr/share/wordlists/dirb/common.txt -e .php,.txt

Step 3: Analyzing Response Codes

Tool chalne ke baad, humein results check karne hain:

  • 200 OK: Direct access mil gaya!
  • 301 Redirect: Folder exist karta hai par humein kahin aur bhej raha hai.
  • 403 Forbidden: Folder hai par humein permission nahi hai (Ye bhi important hai!).

Pro Tip

Agar koi website firewall use kar rahi hai, toh bohot fast scan karne se aapka IP block ho sakta hai. Aise waqt me -t 10 (low threads) ya -p 1.0 (pause) ka use karein.
