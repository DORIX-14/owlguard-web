Understanding the CIA Triad & DoS Attacks

The CIA Triad

Cybersecurity ke teen sabse bade khambe (Pillars).

Confidentiality

Sirf authorized log hi data dekh sakein. (Privacy)

Attack: Data Breach

Integrity

Data ke saath koi ched-chad (alteration) na ho. (Accuracy)

Attack: Data Tampering

Availability

Service hamesha up aur ready rahe. (Uptime)

Target of Lab 13

Lab 13: Denial of Service (DoS)

Jab hum Availability ko target karte hain, toh use DoS attack kehte hain. Iska maqsad data churana nahi, balki server ko "Jam" karna hai.

Step 1: Resource Analysis

Server ki capacity check karein. Ek weak server (jaise 2GB RAM) 10,000 requests mein hi crash ho jayega.

Step 2: Distributed Attack (DDoS)

DDoS mein hum multiple nodes (Botnets) ka use karte hain. Isse target ko lagta hai ki real traffic aa raha hai.
