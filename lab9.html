Step 1: Identify the Hash

Sabse pehle humein ye janana hai ki hash kaunse type ka hai (MD5, SHA1, SHA256). Iske liye hum HashID tool use karte hain.

Logic: Hashing one-way hoti hai. Hum use wapas nahi "decrypt" kar sakte, isliye hum plain text passwords ko hash karke unhe match karte hain.

Step 2: Choosing a Wordlist

Cracking ki success wordlist par depend karti hai. Sabse famous wordlist RockYou.txt hai, jisme millions of leaked passwords hain.

ls /usr/share/wordlists/rockyou.txt

Step 3: Running the Attack

Ab hum Hashcat ka use karenge (jo John se fast hai kyunki ye GPU use karta hai). Example: MD5 hash crack karne ke liye:

hashcat -m 0 my_hash.txt /usr/share/wordlists/rockyou.txt

Security Tip

Weak passwords (jaise 123456) seconds mein crack ho jate hain. Hamesha long aur complex passwords use karein taaki "Brute-force" attack ko saalon lag jayein.
