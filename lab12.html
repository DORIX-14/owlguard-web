Step 1: Site Cloning

Hacker sabse pehle ek login page choose karta hai (e.g., Facebook ya Gmail). SET tool ka use karke wo us page ka HTML/CSS copy kar leta hai.

setoolkit -> 1 -> 2 -> 3 -> 2 (Site Cloner)

Step 2: Hosting the Link

Cloned page aapke local machine par hota hai. Ise victim ko bhejne ke liye Ngrok ya Cloudflare ka use karke ek fake link banaya jata hai.

Logic: Victim ko lagta hai wo asli site par hai kyunki UI ekdum same hoti hai, par URL alag hota hai (e.g., login-secure-update.com).

Step 3: Credential Harvesting

Jaise hi victim apna username aur password enter karta hai, website use save kar leti hai aur hacker ke terminal par bhej deti hai.

POSSIBLE USERNAME: admin | POSSIBLE PASSWORD: password123

Safety Awareness

Phishing se bachne ke liye hamesha URL check karein, 2FA (Two-Factor Authentication) enable karein, aur anjan links par click na karein. Ye lab sirf educational purpose ke liye hai.
